# 05. 마을 홈페이지 & 대한민국 마을 지도 (비회원 열람용)

## 목표

비회원(로그인하지 않은 방문자, 검색엔진, AI 크롤러 포함)이 마을의 활동을 열람할 수 있는 공개 웹페이지.
최상위 진입점은 대한민국 지도이며, 지도 → 마을 홈페이지 → 사진 Feed → Live Chat 순서로 탐색이 이어진다.

## 대한민국 마을 지도 (`/`)

- 지도 라이브러리(예: Google Maps JS API 또는 Kakao Map API 중 택1 — Kakao Map은 국내 정확도가 높아 권장,
  단 카카오 브랜드와 무관한 지도 API로만 사용)
- `communities` 컬렉션 중 `isPublic: true`인 문서를 마커로 표시, `communityType`별로 마커 색상/아이콘 구분
  (부녀회/청년회/노인회/동호회)
- 마커 탭 → `/village/[communityId]`로 이동
- 상단 헤더에 검색(마을 이름 검색), 다크/라이트 모드 토글 스위치, "로그인" 버튼 배치

## 마을 홈페이지 (`/village/[communityId]`)

### 상단
- 마을 이름 + 공동체명 (예: "애월읍 봉성리 부녀회")
- 커버 이미지 또는 최근 대표 사진
- 다크/라이트 토글 (지도 화면과 공유되는 전역 상태)

### 메인 Feed (사진 중심, 세로형)
- 화면에 약 2.5장이 동시에 보이는 세로 슬라이드, 좌우(또는 상하) 스와이프로 넘김
- `communities/{id}/photos`를 `orderBy('createdAt', 'desc')`로 구독, 최신순 자동 정렬
- 사진 위에 업로더 이름, 촬영 위치, 날짜, 기종 오버레이 (04 문서 디자인과 동일 규칙)
- 디자인 톤: Instagram과 Apple Photos의 중간 — 여백을 살리고 사진이 주인공이 되는 레이아웃

### Live Chat 영역 (Feed 하단)
- `communities/{id}/messages`를 실시간 구독해 최근 대화를 노출 (읽기 전용, 비회원은 작성 불가)
- 회원이 채팅 앱(03 문서)에서 대화하면 이 영역에 새로고침 없이 즉시 반영
- "마을 광장 전광판" 컨셉 — 현재 마을 분위기를 보여주는 것이 목적이므로 최근 N개(예: 30개) 메시지만 노출

### 하단/기타 섹션 (선택적으로 배치, 우선순위는 낮음)
- 다가오는 행사 리스트(`events` 컬렉션)
- 게임 결과 하이라이트(최근 `game_result` 타입 메시지)
- 공개 빈집소개 요약(07 문서), 최근 제보 현황(선택적 공개)

## 구현 시 주의사항

- 이 페이지는 SEO/AEO를 고려해 서버사이드 렌더링(SSR) 또는 정적 생성(ISR)을 우선 검토 — AI 크롤러와 검색엔진이
  콘텐츠를 읽을 수 있어야 함 (08 문서의 "AI 친화적 데이터 플랫폼" 목표와 직결)
- 비공개 커뮤니티(`isPublic: false`)는 `/village/[communityId]` 접근 시 "비공개 커뮤니티입니다" 안내와 함께
  콘텐츠 대신 최소 정보(이름 정도)만 노출
- 지도/홈페이지는 로그인 여부와 무관하게 항상 접근 가능해야 하며, 로그인한 회원이 봐도 동일한 화면(단, 상단에
  "채팅 앱으로 이동" 버튼 추가)
